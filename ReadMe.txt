CODE與生成的3Dfigure、image放在個別資料夾中
3Dfigure格式是.fig
image格式是.png
------------------------------------------------------------------------------------------------
part1 images 命名方式是
result_samplePoints_LoD
EX:	
samplePoints = 100	LoD = 0.01
則命名結果為  result_100_001
samplePoints = 100	LoD = 0.2
則命名結果為  result_100_02
-------------------------------------------------------------------------------------------------
part2 3Dfigure命名方式是
(a) --> RGBCube
(b) --> HSV_cyl
(c) --> Q2_c
